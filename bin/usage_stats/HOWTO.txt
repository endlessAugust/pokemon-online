How to usage statistics

1. remove usage statistics plugin
On server:
/unloadstats
-> true

2. Update statistics
On shell:
-> goto server exe directory
LD_LIBRARY_PATH=. ./StatsExtracter_debug
cd usage_stats
python archive_usagestats.py month year
mv raw raw.bk
mkdir raw
./gen_past-stats_index.sh > formatted/Past\ Stats/index.html

3. Reload the plugin
/loadstats
-> true
